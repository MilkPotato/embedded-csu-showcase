日常修改（不消耗积分）：
  git add .
  git commit -m "修改内容"
  git push origin master                         → 推master，Netlify不理

正式发布（消耗15积分）：
  git push origin master                         → 先确保master已推送
  打开 github.com 网站
  点 Pull Requests → New Pull Request
  base: production ← compare: master
  点 Create Pull Request → Merge
  → Netlify 自动部署 ✅
